# Week 2 Portfolio Starter

Open `index.html` in a browser.

Your Week 2 goal is to turn the Week 1 About Me page into a simple portfolio skeleton while practicing a controlled AI workflow.

This is the starting point for Project 1, not the final Project 1 quality bar. The final portfolio should become more polished later through better AI workflow, review, testing, and iteration.

## Tasks

1. Inspect the current `index.html` and `style.css`.
2. Add clear page sections: About, Projects, and Contact.
3. Add two project placeholder cards.
4. Add simple responsive CSS.
5. Test the page in a normal and narrow browser width.
6. Record one meaningful AI interaction in `PROMPTS.md`.

Week 2 success standard: simple, working, explainable.
