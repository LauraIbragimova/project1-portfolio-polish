# AGENTS.md
## Course context
This repository is for AIM 5012: Effective Coding with AI.
The student is learning to use AI while still understanding, testing, and explaining code.

## Workflow
- Inspect the project before editing.
- Explain a short plan before making changes.
- Keep changes small and beginner-friendly.
- Ask before installing dependencies.
- Do not commit unless the student explicitly asks.
- After edits, summarize changed files and verification steps.

## Safety
- Do not read, print, or commit secrets.
- Do not modify files outside this repository.
- Do not run destructive commands.
- Do not paste or store private student data, API keys, passwords, personal IDs, or immigration documents.

## Verification
- Check `git status` before and after work.
- Use `git diff` to review changes.
- Recommend the smallest relevant test or browser check.
